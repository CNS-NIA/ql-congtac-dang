# Hệ thống Quản lý Công tác Đảng

Hệ thống quản lý công tác đảng cho Chi bộ Đội Thiết bị Thông tin Dẫn đường - Đảng bộ Trung tâm Khai thác Khu bay Nội Bài.

## Chức năng chính

1. **Quản lý Chi bộ**
   - Thông tin chi bộ
   - Số lượng đảng viên
   - Lịch sử hoạt động

2. **Quản lý Đảng viên**
   - Hồ sơ đảng viên
   - Quá trình công tác
   - Khen thưởng/Kỷ luật

3. **Quản lý Nghị quyết**
   - Nghị quyết tháng/quý/năm
   - Quy trình phê duyệt
   - Lưu trữ văn bản

## Công nghệ sử dụng

- Frontend: HTML5, CSS3, JavaScript (ES6+)
- UI Framework: Bootstrap 5
- Database: JSON (localStorage)
- Hosting: GitHub Pages
