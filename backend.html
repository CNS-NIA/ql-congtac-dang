API Endpoints

GET
/api/chibo

Lấy danh sách Chi bộ. Có hỗ trợ các tham số phân trang và lọc. 

// Request example
GET /api/chibo?page=1&limit=10&search=
                        
// Response example
{
  "success": true,
  "data": [...],
  "pagination": {...}
}
GET
/api/dangvien

Lấy danh sách Đảng viên (có phân trang). Hỗ trợ tìm kiếm và sắp xếp.

POST
/api/dangvien

Thêm Đảng viên mới vào hệ thống. Yêu cầu xác thực JWT. 

// Request body
{
  "ho_ten": "Nguyễn Văn A",
  "ma_dang_vien": "DV001",
  "chi_bo": "Chi bộ 1",
  "chuc_vu": "Đảng viên",
  "ngay_vao_dang": "2020-05-15"
}
GET
/api/health

Kiểm tra tình trạng server và kết nối database.
